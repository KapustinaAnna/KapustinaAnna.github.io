## Задачи

## Задача 1: Реализовать  экранную форму отображения информации о продукте
**Критерии приемки:**

 - Появление окна с детальной информацией о продукте, включая название, описание, количество, срок годности.

### Основной  сценарий

|**Шаг**|**Описание шага**|**Результат**|
|-------|---------------------------------------------------------------|----------------------------------------------------|
|1 |Открыть приложение|Пользователь видит главный экран приложения.|
|2 |Пользователь отсканировал штрихкод|Система успешно считывает штрихкод и отображает информацию о продукте|
|3 |Просмотреть информацию о продукте|Окно с детальной информацией о продукте появляется на экране.|
|4 |Проверить, что отображается название продукта|Название продукта соответствует данным в базе.|
|5 |Проверить, что отображается данные о продукте: срок годности, категория, количество, поставщик, склад.|Описание продукта соответствует данным в базе.|
|6 \.|Пользователь нажимает на кнопку закрыть|Окно с информацией о продукте успешно закрывается|

-----

### Негативный сценарий

|**Шаг**|**Описание шага**|**Результат**|
|-------|---------------------------------------------------------------|----------------------------------------------------|
|1 |Открыть приложение|Пользователь видит главный экран приложения.|
|2 |Ввести несуществующий штрихкод |Система не находит продукт и отображает сообщение об ошибке.|
|3 |Ожидать, когда появится сообщение об отсутствии продукта|На экране появляется сообщение: "Продукт не найден."|

## Задача 2: Реализовать функционал  системы для поиска товара по штрихкоду
**Критерии приемки:**

  - Система должна принимать входящие сигналы от аппарата для сканирования.
  - При успешном сканировании система должна отображать информацию по продукту.

### Основной сценарий

|**Шаг**|**Описание шага**|**Результат**|
|-------|---------------------------------------------------------------|----------------------------------------------------|
|1 |Открыть приложение в режиме инвентаризации|Приложение успешно открыто|
|2 |Навести сканер на штрихкод продукта|Система получает сигнал на сканирование|
|3 |Прочитать штрихкод|Система успешно считывает штрихкод и отображает информацию о продукте|
|4 |Просмотреть информацию о продукте|Пользователь видит подробные данные о продукте, включая название, цену, описание и наличие|

-----


### Негативный сценарий

|**Шаг**|**Описание шага**|**Результат**|
|-------|---------------------------------------------------------------|----------------------------------------------------|
|1 |Открыть приложение в режиме инвентаризации|Приложение успешно открыто|
|2 |Навести сканер на штрихкод|Система не распознает штрихкод и выводит сообщение об ошибке|
|3 |Попытаться просмотреть информацию о продукте после неудачного сканирования|Пользователь видит уведомление о том, что информация о продукте недоступна или сканирование не удалось|

## Задача 3: Реализовать функционал обработка ситуации, когда продукт не найден.  
**Критерии приемки:**

  - Система должна выводить сообщение об ошибке, если продукт с заданным штрихкодом не найден.
  - Пользователь должен иметь возможность ввести  информацию о ненайденном продукте.

### Основной сценарий

|**Шаг**|**Описание шага**|**Результат**|
|-------|---------------------------------------------------------------|----------------------------------------------------|
|1 |Открыть приложение|Пользователь видит главный экран приложения.|
|2 |Навести сканер на штрихкод продукта|Система получает сигнал на сканирование|
|3 |Прочитать штрихкод, который отсутствует в базе данных|Система отображает сообщение: "Продукт по указанному штрихкоду не найден."|
|4 |Нажать кнопку "Добавить продукт"|Пользователь видит форму для ввода информации о новом продукте|
|5 |Ввести информацию о новом продукте|Пользователь успешно заполняет все необходимые поля.|
|6 |Нажать кнопку сохранения информации|<p>Отправлен запрос на добавления продукта в БД.</p><p>Появилось сообщение, что продукт успешно добавлен в корзину</p>|

-----

|**Шаг**|**Описание шага**|**Результат**|
|-------|---------------------------------------------------------------|----------------------------------------------------|

|2 |Навести сканер на штрихкод продукта|Система получает сигнал на сканирование|

|6 |Нажать кнопку сохранения информации|<p>Отправлен запрос на добавления продукта в БД.</p><p>Появилось сообщение, что продукт успешно добавлен в корзину</p>|
